##pictureBox
 - 可以显示来自位图、图标或者元文件。以及来自增强的元文件、JPEG、GIF文件的图形。如果控件不足以显示整幅图像。则裁剪图像以适应控件的大小（sizeMode属性）
 - normal 裁剪图片
 - stretchIamge 图片调整大小适应控件
 - AutoSize 控件适应图片
 - centerImag 图片处于控件中心
 - zomm 图片调整大小适应控件。控件宽高比不变
 -  代码获取image图片`pictureBox1.Image = Image.FromFile(@"文件路径")`；
